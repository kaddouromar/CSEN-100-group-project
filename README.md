## About nearU.
Tired of hearing "there's nothing to do in Cyprus"? We're changing that. From hidden study spots and local food gems to campus events and beach activities, find everything students actually love doing in and around AUB Mediterraneo.

<br />

## Features

| Feature                            | Details                                                                                                                                                                       |
|------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Interactive Map                    | A live map displaying all user visited locations with their respective ratings. The colour of each waypoint determines the type of activity each activity is classified under |
| Account Creation                   | Fast, easy and simple account creation                                                                                                                                        |
| Easy Event Creation/ Participation | Any user can post an event                                                                                                                                                    |
| Reward System                      | Users gain XP points when pinning locations, offer recommendations and participate in events                                                                                  |

<br />

## Setup Instructions

1. Clone the repository in your desired directory using the following command: `git clone git@github.com:kaddouromar/CSEN-100-group-project.git`

2. Navigate to this directory

3. Install dependencies using: `npm install`. if npm is not installed on your system, follow [this article](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm "npm and node.js guide")

4. Time for the fun part! Run the server by using this command in your terminal: `npm run dev`

5. Enjoy!

## Success Metrics

* User Traction: The amount of people who use the website (measured on a daily basis)
* Number of Events: There should be multiple consecutive events happening every month
* Sponsorships: In order to keep a consistent flow of users, we would sponsor events to gain more users

## Risks & Solutions

* Security: Possible threat actors may attempt to break the website
	* Solution: Sanitisation of all forms of user input

* User Adoption: Real-time social apps may feel slow at first due to lack of community
	* Solution: Through the money saved by using AWS, we could throw more money towards advertising.

* Safety & Privacy: PII data would need to be thoroughly protected
	* Solution: Correct use of GDPR & CCPA Policies
